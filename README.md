# ☕ Coffee Shop Sales Dashboard | Excel Project

## Turning Raw Data into Business Decisions

What if a coffee shop could instantly know:

* When customers buy the most?
* Which products actually drive revenue?
* Which store performs the best?

This project answers those questions using **only Microsoft Excel** — no coding, no SQL, just smart analysis and visualization.

---

## About the Project

I built an **interactive sales dashboard** using real transactional data to uncover patterns in:

* Customer buying behavior
* Product performance
* Time-based sales trends
* Store-wise revenue comparison

The goal was simple: **make data easy to understand and useful for decision-making.**

---

## Key Features

✔ Interactive dashboard with slicers
✔ Clear KPI tracking (Revenue, Orders, Quantity)
✔ Hourly, daily & monthly trend analysis
✔ Top-selling products identification
✔ Store performance comparison

---

## What Makes This Project Strong

Instead of just showing data, this dashboard helps answer real business questions:

* When should staff be increased?* → Peak hours identified
* What should we promote more?* → Top products highlighted
* Which store needs attention?* → Location comparison
